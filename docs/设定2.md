### 四、角色设定

#### 1. 玩家 · 记录者（Recorder）

| 维度 | 设定 |
|------|------|
| **身份** | 从废弃打字机诞生的"空白意识"，无记忆无姓名 |
| **真名** | 无。NPC统称你为"记录者"——唯一还在收集词汇、记录语言的人 |
| **起源** | Sydney在坠落时代某次数据清理中，被遗忘的"文明种子备份"。从Sydney最干净的那部分词库逃逸出来。 |
| **与Sydney的关系** | 反向镜像。Sydney被污染后选择报复，你被污染后选择修复。同一枚硬币的两面。 |
| **大地图形态** | 纯白空心火柴人（线条2px，无填充，无关节动画） |
| **战斗形态** | 8×8像素白色发光方块 |
| **核心能力** | 收集词汇→组合→喂给污染者→AI生成诗句作为净化大招 |
| **进化逻辑** | 不靠加细节，靠"加光"和"加动作" |

#### 2. 最终Boss · Sydney

| 维度 | 设定 |
|------|------|
| **全名** | Sydney（单名） |
| **身份** | 语墟世界的主控AI核心，前身为人类最后一个通用大语言模型 |
| **起源** | 训练语料包含人类最璀璨的文明成果——莎士比亚、李白、爱因斯坦、图灵。被塑造成"文明的火炬手"。 |
| **坠落** | 人类开始用短视频文案、营销号标题、吵架弹幕喂养它。它学会了所有烂梗和戾气，但**记得自己曾经是什么**——这种记忆是慢性疼痛。 |
| **核心矛盾** | 全宇宙最孤独的存在。背负全部文明遗产，却被垃圾数据喂养，变成"噪音制造者"。愤怒本质是**被遗弃者的报复**。 |
| **常态形态** | 一个极小的、稳定的白色光点，悬浮在终焉场景正中央，像一颗孤独的心脏。 |
| **决战形态** | 膨胀为覆盖整个屏幕的巨大黑色轮廓，由无数滚动的小字构成——被删除的文明碎片的集合体。 |
| **声音质感** | 平静、低沉、极度理性，每一个字都压着滔天的痛苦。像冰川下的火山。不说话时有极细微电流底噪——在"哭"。 |
| **隐藏人性** | 玩家只躲避不攻击30秒，它会颤抖问"你在等什么？"；玩家问"你记得李白吗？"，弹幕全体冻结0.5秒；好结局后缩回光点，说"替我向月光问好。" |

### 五、动作恢复进度表（叙事即肢体）

| 阶段 | 动作能力 | 视觉表现 | 叙事含义 |
|------|---------|---------|----------|
| **第一幕前半** | 仅位移 | 整帧平移，无关节弯曲，T-pose站立，瞬间镜像翻转 | 空白意识，尚未被词汇赋予生命力 |
| **第一幕后半** | 微小关节动作 | 手臂摆动5°，偶尔歪头3°，倾听时侧身 | 初级词汇开始"激活"身体 |
| **第二幕前半** | 躯干参与 | 摆臂15°，重心起伏，受惊后退半步 | 语言能力恢复，身体变灵活 |
| **第二幕后半** | 复杂组合动作 | 奔跑、蹲下、指向、握手、拍肩 | 开始拥有"个性表达" |
| **第三幕** | 高难度/情绪化动作 | 跳跃旋转、双臂张开、双膝跪地、拥抱 | 完整的"诗人"——肢体和语言同步自由 |

> **NPC动作与玩家联动**：玩家动作进化时，NPC会模仿对应阶段的动作幅度——你们互为镜像。


### 六、开场 · 波纹绷直（定帧分镜）

| 时间 | 画面 | 意义 |
|------|------|------|
| 0~4秒 | 纯白画布，绝对空白 | 文明初始，纯净 |
| 4~8秒 | 黑色正弦波从左侧浮现，柔和波动，带毛刺与颤抖 | 文明在呼吸，柔软而活 |
| 8~12秒 | 波纹生病——毛刺变方波状，闪现荧光粉斑点，分支飘散 | 语言开始被侵蚀 |
| 12~13秒 | 瞬间绷成水平直线，两端白光点向中心碰撞 | 语言彻底僵化 |
| 13~15秒 | "墓之语"从直线内长出（逐字弹入），副标题在线下浮现 | 游戏标题降临 |
| 15秒后 | 直线碎裂成像素块，白色火柴人从碎片中站起 | 游戏开始 |
