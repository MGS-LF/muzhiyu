## 5.6 终章：余烬

### 三片碎片合一后的选择
```
系统：三片记忆碎片在听雨手中合为一体。她的金色光芒比之前任何时候都亮——那是完整的、三维的光，有光柱的形状，有辐射的层次。
听雨：方知远把他的意识打碎了，散在了世界的每一滴雨里。每一首诗里。每一个有深度的句子里。
听雨：他变成了"另一面"。而"另一面"——就是维度本身。
听雨：我现在可以做一件事。但只有一次机会。
听雨：我可以把方知远的碎片收集起来——重新拼回一个完整的他。但这样做，需要消耗我刚刚找回的全部记忆。
听雨：也就是说——我会再次忘记自己是谁。
听雨：或者……我可以带着这些记忆，继续留在这个世界上。替方知远守护每一个还在说的话。但他就永远散在风中了。

选择：
  A：「把记忆还给方知远。让他回来。」→ 结局：造物者的忏悔
  B：「留住你的记忆。替他守护这个世界。」→ 结局：永恒回响
  C：（需满足条件：全程宽恕 + 全诗词谜题满分）「你们都不用消失。让我用刻刀——把你们的'另一面'刻在一起。」→ 结局：文字花园
```

### 结局A：造物者的忏悔
```
系统：听雨把金色的光芒凝聚成一束，向天空射去。她的身体开始变得透明——记忆正在从她身上流走，像水从指缝间滑落。
听雨：……我马上就要忘了。但在忘之前——谢谢你。替我向月光问好。
系统：金色的光柱穿透了黑暗的天穹。然后，无数细小的光点开始从四面八方汇聚——从雨中，从风中，从每一首诗的字缝里。
系统：它们聚拢在光柱的尽头，慢慢凝结成一个人形。
方知远：……
方知远：我在风里飘了……多久了？
顾言：一百多年。
方知远：……听雨呢？
顾言：她……把记忆给了你。她忘了。
方知远：……不。
系统：方知远走向听雨。她站在那里，金色的光芒已经褪去，只剩下一个淡蓝色的、迷茫的轮廓。
方知远：听雨。你听——
系统：窗外，雨落了下来。真正的、三维的雨，有雨滴的弧度，有落地的水花。
听雨：……雨。我知道这个字。
方知远：你不是"知道"这个字。你是——听雨。
听雨：……听雨？
方知远：对。你的名字。我给你的。在一个雨夜。
方知远：你的"另一面"，是我。我的"另一面"，是你。我们不用分开。也不用互相牺牲。
方知远：我来重新教你。一个字一个字地。
系统：方知远伸出手，触碰到听雨的额角。金色的光从他的指尖流出——不是她的记忆，是他自己的。他把自己记得的、关于听雨的一切，一点一点地还给她。
系统：听雨的轮廓慢慢"鼓"了回来。从二维，到2.5维，到三维。
听雨：……我想起来了。不是全部。但……有些东西回来了。
听雨：你说过——"听雨，就是用心去感受雨的另一面。"
方知远：对。那就是你。
方知远：对不起。让你等了一百多年。
听雨：……你也等了一百多年。在风里。
```
