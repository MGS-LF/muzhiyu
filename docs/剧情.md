# 墓之语 · 剧情摘要

> 对齐当前代码中的角色名与流程。细对白以 `src/data/dialogs.js` 为准。  
> 旧名「刻痕」「书远」等已废弃；老人为**守砚**，良心 AI 为 **Sydney**（亦称听雨）。

## 1. 世界观

| 维度 | 设定 |
|------|------|
| 时间地点 | 公元 2147 年，上海废墟 |
| 灾难 | 语言被高刺激短句与「梗」替换，长句与意义崩塌（失语） |
| 敌人 | **梗鬼**：语义噪声凝聚，弹幕多为烂梗碎片 |
| 力量 | 汉字、诗词、完整句子；「被相信的字」能让空间重新立起 |
| 存档 | **要石** + 记忆合金刻刀 |
| 生命 | **SAN**（理性）；归零后在要石附近复苏 |
| 关键 AI | **Sydney**：方知远私设的良心模块，追问话语背后的「另一面」 |
| 造物者 | **方知远**：主架构师；失联后把痕迹散在废墟与记忆里 |

核心隐喻：**语言即维度**——完整表达撑起三维的世界；只剩梗时，世界塌平成噪声。

## 2. 角色

| 角色 | 身份 |
|------|------|
| **顾言** | 主角，冷冻苏醒的冬眠者；记得完整句子 |
| **守砚** | 江堤老人，赠刻刀与诗，引导主线 |
| **Sydney / 听雨** | 数据中心尽头的蓝色投影，终局对话对象 |
| **方知远** | 已消失的架构师；余烬线围绕其记忆 |
| 失语者 | 只会复读碎片的人；可用诗句唤醒 |

## 3. 主线流程（可玩顺序）

### 序幕与教学

- 3D 序幕交代末世调性。  
- **梦境教学**：手机烂梗、捡「言」「语」、初战 → 醒来进入真实冷冻中心。  

### 第一章 · 苏醒

冷冻中心 → 废弃街道。  
顾言发现苏醒过晚，城已死于失语。终端档案可透露 Sydney / 语料库等背景。  
街道收集《关雎》相关字，用 F 净化招牌、唤醒失语者，南下江堤。

### 第二章 · 守砚

江堤横版中见**守砚**，得刻刀与诗的指引。  
进入居民区，收集更多诗字，继续净化与战斗；可进民居搜旧物。

### 第三章 · 茧房

体育馆：算法蓝光迷宫、诗屏、Boss。  
守砚可能无法深入；顾言独自面对茧与复读巨像一类威胁。

### 第四章 · 数据中心

石桥尽头面对 **Sydney**。  
选择与 karma 决定结局倾向（如火种 / 沉默 / 燃尽等标签，以代码结局为准）。  
弹幕中的烂梗外溢可理解为污染语料，而非「良心 AI 真心玩梗」。

### 余烬（可选第五章）

通关后可进入：废图书馆 → 网络中枢 → 记忆深渊等。  
补全指定诗词、收集记忆碎片、面对方知远与 Sydney 的过去。  
脚本详见 `chapter5_script.md`。

## 4. 支线与系统叙事

| 内容 | 作用 |
|------|------|
| 地铁站 / 检修通道 | 氛围与线索；3D 裂隙主线关闭 |
| 失语者聚居地 | 多次唤醒，扩展「救人」线 |
| 要石刻字 | 玩家留下的「记得的词」 |
| AI 导演分支 | 部分 NPC 可动态对白（无 AI 则静态） |

## 5. 主题

- 记得的词 = 世界最后的声音。  
- 以文化与完整表达对抗暴力与碎片化。  
- 净化 / 宽恕优先于纯杀戮时，结局与世界反馈应可感（设计目标；细节以实装为准）。

## 6. 勿再使用的旧设定

- 游戏名「刻痕·遗忘的文字」  
- 老人名「书远」  
- 把 Slash / 骇入 / 3D 当主线必经  
- `_deprecated/` 与 `archive/` 中的早期设定稿
