## 1. 介绍里真正要兑现的是什么

介绍把游戏说成一剂**语言困境的文化解药**，不是「多模式展示」。

### 1.1 必须让玩家玩到的（成功标准）

| 介绍主张 | 游戏里必须发生的事 |
|----------|-------------------|
| 大失语末世 · 顾言苏醒 | 冷冻→废墟，调性统一「语言死了」 |
| 收汉字 → 拼诗词 | **日常主循环**，不是偶尔开门 |
| 弹幕打**梗鬼**（吞噬语义） | 敌与弹幕可读为「梗」；诗/净化能克它 |
| 选择决定：语言能否被记起 | karma + 探索深度 → 多结局可感 |
| 语言即维度 | 净化前后世界「扁/立」肉眼可见 |
| 以文化对抗暴力 | 净化/宽恕优于纯砍的反馈与结局 |
| 三映射：净化梗鬼 / 唤醒失语 / 复原诗词 | 主线里各反复出现，有独特反馈 |
| 诗 · 剑 · 衡 | 收集复原 / 净化战斗 / 道德结局 三条线咬合 |
| Slogan：记得的词 = 世界最后的声音 | 词袋/图鉴/刻痕/终局回放能接住 |

### 1.2 介绍里写了、但会毁掉「解药感」的（游戏侧收敛）

介绍曾并列 2D + 横版 + 3D FPS。对**初衷**而言这是噪音：

- 操作每换一套，玩家就从「琢磨字与话」变成「学新键位」  
- 与「对抗碎片化专注」相反  

**游戏改造裁决（不改 PPT 文案）：**

- 主线只保留：**俯视角探索 + 组句/诗（F）+ 关键弹幕**  
- 3D / 黑客 / Slash：主线入口关闭（`FEATURES`），可留调试  
- 横版：**江堤**保留为正式横版关（跳跃 / 挥刀），不扩散到其它主线场景  

目标不是否定介绍，而是让**实际游玩**对准介绍里的社会意义与三柱玩法。

### 1.3 一句话（开发用）

> 玩家每几分钟都在用更完整的话，对抗只会玩梗的世界；打和模式切换都是配角。

## 2. 目标手感

### 2.1 主循环（2～5 分钟）

```
看见梗化（复读的人 / 热梗墙 / 梗鬼）
  → 捡汉字
  → 组短句，或推进一整首诗
  → 净化 / 宽恕 /（必要时）攻击
  → 人恢复表达、场景「立起」、鬼散成字
  → 道德与结局悄悄偏移
```

爽点必须**当场看得见**，不能只改内部 flag。

### 2.2 操作上限（防杂）

1. 移动 + E 交互  
2. 组句 / 诗相关 UI  
3. 弹幕战：移动 + 攻击/净化/宽恕  

### 2.3 体验占比

| 占比 | 内容 | 对应介绍 |
|------|------|----------|
| ~70% | 收字、组句、唤醒、改招牌/地形 | 诗 + 映射 2/3 |
| ~15% | Boss 弹幕；净化与宽恕改阶段 | 剑 + 映射 1 |
| ~15% | 短对白、要石、结局回响 | 衡 + slogan |

### 2.4 三柱在游戏内的咬合

```
诗：单字 → 词袋能说的话 → 整诗 → 大招
剑：探索劝退 → 小战 → Boss 弹幕（弹幕=梗，破招=诗）
衡：打散 vs 把话还回去 → karma → 多结局
```

## 3. 章节里「语言课题」要可玩

| 章 | 名 | 课题（玩家应感到） | 主行为 |
|----|----|--------------------|--------|
| 1 苏醒 | 你还记得字吗？ | 捡字、第一次把话说完整 |
| 2 烂梗 | 梗如何吃掉语义？ | 梗墙、杂兵净化、失语者、中型战 |
| 3 茧 | 算法如何锁住所见？ | 轻潜行 + 茧后（净化优先） |
| 4 深处 | 语言还能被记起吗？ | Sydney、终战、三结局 |
| 5 余烬 | 记起后如何补全？ | **短**尾声三记忆，不新开沙盒 |

**体量：** 主线必经场景求精（约 8～10），不靠堆图体现「大作」。  
失语者 **≥5** 且每人不同「缺词 + 醒来的一句完整话」。  
硬诗主线 **6～8** 首有战斗/场景回报；图鉴可更多。

### 3.1 结局（游戏内可测即可）

| 结局 | 方向 |
|------|------|
| 火种 / 沉默 / 燃尽 | 第四章主选择 + karma |
| 忏悔 / 回响 / 花园 | 余烬与探索/宽恕深度 |

具体 flags 在实现 P4 时写成表驱动，保证能调试触发。

## 4. 现实映射 → 关卡义务

| 行为 | 现实意味 | 游戏义务 |
|------|----------|----------|
| 净化梗鬼 | 抗粗鄙化、单一化 | 弹幕/外形是梗；净化比纯砍更「世界变好」 |
| 唤醒失语者 | 重建表达与共鸣 | 缺词解谜；醒来必须是**完整有情绪的话** |
| 复原古诗 | 文化生命力 | 整诗 → 大招或破门/破 Boss，不只有勾选 |

## 7. P0 — 诗柱样板（先做）

### 7.1 目标

`street_01`，约 5 分钟：

1. 捡 ≥3 字（强反馈）  
2. 组句  
3. 净化梗墙和/或 1 名失语者  
4. 立刻变世界  

### 7.2 样板实体

| ID | 映射 | 成功 |
|----|------|------|
| `meme_wall_01` | 抗粗鄙化 | 热梗→正常路名/半句；路更「立」 |
| `aphasic_01` | 共鸣 | 复读梗→一句真情完整话 |

- 上限 6（逼取舍，贴「词汇匮乏」）  
- P0 可先：图鉴收集 + 释放记场景 flag

### 7.4 预期改动文件

`config.js`、`player.js`、`systems/utterance.js`（新）、`interact.js`、`game.js`、`scenes.js`、`render/hud|panels`、`data/dialogs.js`、`save.js`

### 7.5 验收

- [ ] 不战斗也能完成一次映射向净化  
- [ ] 缺字/错字有提示  
- [ ] 成功：声 + 光 + 文案/碰撞变化  
- [ ] 存档不丢进度  
- [ ] 不冲掉 E 对话/要石/原战斗入口

## 17. 总验收（对照介绍精神，验游戏）

1. 玩家会用「汉字、诗词、梗鬼、记不记得语言」描述本作  
2. 三映射在关卡里各举得出例子  
3. 净化/宽恕与纯暴力，结局或世界反馈不同  
4. 主线 ≤3 套操作；每约 2–3 分钟有一次「说出 → 世界变好」  
5. 开场或终局接得住：「你记得的词…」/「不要让最后一个词死在沉默里」  
6. 无 AI 可通关  
7. 默认主线 **0 次** 3D / 骇入 / 长横版
