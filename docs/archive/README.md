# 归档说明

本目录为**历史文档**，不作为开发依据。

| 文件 | 原用途 | 为何归档 |
|------|--------|----------|
| `GAME_ANALYSIS.md` | 旧实现分析 | 场景/系统/开关已过时 |
| `game_design.md` | 大设计稿 | 键位与模式描述落后于代码 |
| `game_script.md` | 脚本草案 | 场景数据以 `src/scenes.js` 为准 |
| `competition_*.md` | 赛事材料 | 非运行时文档 |
| `纯剧情阅读版.md` | 剧情连读 | 角色名/流程可能旧；见 `剧情.md` |
| `新手引导方案.md` | 引导设计 | 实装见 `onboarding.js` + `CURRENT.md` |
| `新手引导重构方案_线性梦境.md` | 引导重构稿 | 同上 |

更早废弃设定：`docs/_deprecated/`。
