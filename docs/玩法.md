# 墓之语 · 玩法说明

> 对齐当前代码。键位以 `src/data/controls.js` 与根 README 为准。

## 1. 你在玩什么

大失语之后的废墟里，完整的句子正在消失。你要：

1. **捡汉字**（地上碎片、战斗掉落）  
2. **补诗**（靠近梗墙 / 失语者按 **F**）  
3. **战斗**（接触梗鬼 → 弹幕菜单战）  
4. **选择**（净化 / 宽恕 / 攻击影响 karma 与结局）  

隐喻：烂梗把世界「压扁」；被记住的字与诗能让场景重新「立起来」。

## 2. 日常探索（主时间）

| 行为 | 操作 | 反馈 |
|------|------|------|
| 移动 | WASD / 方向键，Shift 跑 | 俯视角地图 |
| 互动 | E | 对话、拾取、推门、终端 |
| 补诗 | 靠近发光招牌或失语者 → **F** | 填空用已收集字；成功净化 / 唤醒 |
| 目标 | 看左上角 / Q 任务 | 金色箭头指向当前目标 |
| 地图 | M 世界图，Tab 小地图 | 小地图有战争迷雾 |

### 收字与门禁

- 字进入「当前可用」与「永久收集」两套记录（战斗弹药感 + 开门进度）。  
- 去下一区域常需集齐某几字，或完成 compose 诗谜。  
- 街道偏《关雎》；居民区偏《滕王阁序》/《正气歌》；体育馆偏《春晓》；余烬另有诗套。

### 要石与刻字

- 要石：存档 / 复活点。  
- 刻字：留下自定义文字；可选 AI 点评（无密钥则跳过）。

## 3. 补诗净化（F / Utterance）

- 交互类型多为 `purify`（梗墙 / 失语者）。  
- UI：用已有汉字填空，顺序匹配。  
- 成功：场景反馈、flag、karma（如 mercy / saved）、可能触发对白。  
- 与「诗屏 compose 谜题」共用汉字，但是另一套界面。

## 4. 战斗

### 主战斗：UT 弹幕菜单（`Battle`）

- 接触梗鬼进入。  
- 菜单：**攻击 / 调查 / 净化 / 宽恕**（以界面文案为准）。  
- 弹幕阶段移动红心躲避；攻击瞄准用 E / Space 停光标。  
- **K**：诗词大招（需完整诗句，每场限制见实现）。  
- 部分敌人特殊机制（如格式化者影响碎片）。  

### 江堤横版（`SideScrollLevel`）

- 场景：`riverside`。  
- 跳跃、**J 挥刀**、踩踏梗鬼；见守砚后可进居民区。  
- 死亡等会回到街道要石附近（见 `exitSidescroll`）。

### 默认关闭（勿当主线）

Slash 战、骇入战、地铁 3D 裂隙：代码可留，主线 `FEATURES` 关闭。

## 5. 节奏结构（体验占比，设计意图）

| 占比 | 内容 |
|------|------|
| 大部分 | 探索、收字、F 净化 / 唤醒、读环境 |
| 约一成半 | 关键弹幕与 Boss |
| 约一成半 | 对白、要石、结局与选择 |

## 6. 章节手感（主线）

| 阶段 | 场景 | 玩家主要在做什么 |
|------|------|------------------|
| 教学 | 梦境 | 学会移动、捡字、开战 |
| 苏醒 | 冷冻中心 | 换装出门 |
| 废墟 | 街道 / 地铁支线 | 收字、F、初战 |
| 守砚 | 江堤 → 居民区 | 横版 + 继续主循环 |
| 茧房 | 体育馆 | 迷宫、诗屏、Boss |
| 终局 | 数据中心 | 面对 Sydney / 结局 |
| 可选 | 余烬 / 失语村落 | 记忆诗谜与扩展结局 |

## 7. 难度与存档

- 难度影响敌人生命、弹幕、SAN 等。  
- 自动存档 + F6 手动槽 + F4/F9 快存快读。  
- 梦境教学期间一般不写入正常进度污染主线。

## 8. 移动端

虚拟摇杆 + E / K / Space；补诗与菜单以屏幕提示为准。
