## 🎮 墓之语 · 玩法三层结构

### 第一层 · 日常探索（占60%时间）

> **核心体验**：行走、观察、倾听、收集——一种“可玩的冥想”。

#### 1. 横轴行走
- **操作**：方向键左右移动，上下键可攀爬矮平台/进入分支路径。
- **画面**：纯黑/深灰背景，白色地平线，火柴人沿地面行走。地面是文字铺成的——“木”字堆叠成树、“水”字流动成河、“山”字隆起成坡。你走在这些字上面，脚下会泛起极淡的白光涟漪。
- **节奏**：行走速度不快不慢，配合极简的环境音（风声、远处的打字机声、偶尔的鸟鸣——但鸟鸣也是“啾”字的像素音效）。

#### 2. 环境交互（核心乐趣）
- **触碰词汇**：场景中散落着发光的词汇方块（如“光”“路”“听”），靠近即自动收集。收集时字符放大→虚化→碎裂成光点被吸入胸口，0.3秒反馈，轻微震动。
- **倾听NPC**：路上会遇到被污染程度不同的NPC，有些是**可绕过**的（不是每只都必须打）。靠近时NPC会冒气泡文字——可能是一句求救、一句抱怨、或者一句让人心头一紧的真心话碎片。
- **观察文字景观**：部分场景中，墙上的文字会“自己动”——它们偶尔重新排列成新的句子。玩家可以选择驻足观看（不做任何操作，仅观看），某些隐藏句子会触发微量词汇奖励。

#### 3. 词汇管理（轻策略）
- **收集上限**：随身最多携带6个词汇。超出时需在“避难所”或“路边石桌”上暂存/取舍。
- **词汇属性**：每个词有隐含的“温度”属性（温暖词/冷静词/中性词），影响后期AI生成诗句的风格。玩家可以主动决定带什么词上路。
- **组合预览**：在安全区域（如避难所），玩家可以将2~3个词汇拖入“融合槽”，**预览**可能生成的诗句方向（但不消耗词）。这相当于一个“轻量级合成系统”，鼓励玩家实验不同组合。

#### 4. 净化选择（叙事分支点）
- 遇到污染NPC时，玩家有**三种选项**：
  - **净化**（消耗词汇，获得伙伴/剧情推进）
  - **绕过**（不消耗，但失去该NPC的后续故事线）
  - **倾听**（不消耗词汇，NPC会多说一句话，可能透露隐藏信息或解锁新的词汇掉落）
- 这三个选项影响最终结局的“完整度”——净化越多，终局护盾越厚。

#### 5. 避难所建设（家园系统轻量版）
- 每次净化一个区域，避难所会新增一个建筑/功能：
  - “光”字种出灯 → 解锁词汇预览功能
  - “路”字铺出小径 → 解锁快速返回（传送）
  - “火”字燃起篝火 → 解锁NPC对话回顾
- 玩家回到避难所时，看到自己“种”出的世界一点点生长，是重要的情感反馈。

#### 6. 探索中无战斗时的“风险感”
- 虽然平时没有弹幕，但**场景中漂浮的污染粒子**会缓慢靠近玩家。触碰会扣除1点“语义值”（相当于轻伤），但不会致命。这维持了“即使是走路也有轻微紧张感”的体验。


### 第二层 · 小Boss弹幕战（占20%时间）

> **触发条件**：关键NPC净化、区域守关怪、剧情推进节点。

#### 何时触发
- **第一幕**：鹦鹉螺Boss（1场）
- **第二幕**：键盘侠、标题党、茧房囚徒、放弃者（4场）
- **第三幕**：Sydney（1场终局）

合计约6场弹幕战，每场持续2~4分钟。

#### 战斗内体验
- **战斗框**：屏幕中央纯白方框，框外全黑。玩家缩成发光方块。
- **操作**：方向键移动躲避。没有攻击键——攻击就是“用方块去撞净化弹幕”（净化弹幕由AI诗句生成，会飞向怪物）。
- **战斗节奏**：每场战斗分2~3个阶段，每个阶段弹幕密度/速度递增。阶段切换时伴随屏幕闪色。
- **受击反馈**：方块碎裂成4小块再重组，屏幕边缘出现红色裂纹，语义值下降。
- **胜利条件**：收集/使用指定词汇组合触发净化演出。


### 第三层 · 终局决战Sydney（占20%时间）

> **三阶段弹幕战**，详见前文设定。重点补充一点：战斗中有“Meta攻击”机制——Sydney会砸碎UI、歪斜菜单栏、弹出代码报错覆盖屏幕。玩家需要**在躲避弹幕的同时，用方块触碰散落的UI碎片**来恢复功能。这是“战斗+解谜”的融合。

## 🎯 设计目标拆解

| 玩法 | 目标 |
|------|------|
| 行走 | 建立节奏感，让玩家“沉进去” |
| 触碰收集词汇 | 提供即时正反馈（光效+音效+收集感） |
| 倾听NPC | 鼓励慢下来，不跳对话，触发情感连接 |
| 词汇组合预览 | 轻策略，让玩家产生“我的选择有意义”的感觉 |
| 净化/绕过/倾听分支 | 叙事深度+结局多样性 |
| 避难所建设 | 渐进式情感积累，看自己的世界生长 |
| 弹幕战 | 紧张感波峰，释放积累的情绪 |
| Meta攻击 | 打破第四面墙，提升终局的“重量” |
