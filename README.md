# 墓之语

《墓之语》是一款 HTML5 Canvas 2D 废墟探索 RPG。玩家扮演从冷冻仓醒来的顾言，在 2147 年的上海废墟中收集诗词碎片、对抗烂梗污染、唤醒失语者，并追寻良心 AI「Sydney」与方知远留下的记忆。

主线以**俯视角探索 + 捡字补诗（F）+ 关键弹幕**为主：净化梗墙、唤醒失语者、复原诗词，在数据中心面对多结局。江堤为俯视角；3D 裂隙 / 长横版 / 骇入战默认关闭（`src/config.js` 的 `FEATURES`）。通关后可选「余烬」。另含刻字、存档、难度、小地图、移动端触控、诗词大招、AI 对话与 TTS（可降级）。

## 游戏内容

- **主线探索**：冷冻中心 → 废弃街道 → 黄浦江江堤 → 居民区 → 体育馆茧房 → 数据中心。
- **核心循环**：捡汉字 → 按 F 补诗净化/唤醒 → 世界反馈；关键战斗用弹幕 + 攻击/调查/净化/宽恕。
- **诗词谜题**：复原《关雎》《春晓》等，开门禁或削弱 Boss；F 组句与 compose 谜题共用汉字收集。
- **余烬（可选）**：通关后可进废图书馆等，找回记忆碎片与扩展结局。
- **刻字与要石**：复活点与自定义刻痕；AI 可在结局汇总刻字。
- **支线**：失语者、旧书页、语言种子、地铁站等。
- **移动端**：虚拟摇杆与 E / Space / K 触控按钮。

## 运行方式

需要 Node.js 18 或更高版本。

```bash
npm install
```

开发模式：

```bash
npm run dev
```

该命令会同时启动 Node AI/静态服务器和 Vite 开发服务器。通常打开 Vite 输出的本地地址即可。

试玩/服务端模式：

```bash
npm start
```

默认地址为：

```text
http://localhost:8080
```

也可以使用兼容入口：

```bash
node server.js
```

修改端口：

```bash
PORT=9000 npm start
```

## AI 接入

完整体验需要同源 Node 服务提供 AI 代理：

- **LLM 导演**：关键剧情分支和 Sydney 自由对话使用 DeepSeek 接口。
- **TTS 配音**：对白可通过小米 MiMo TTS 生成语音。
- **语音缓存**：生成后的音频会缓存到 `cache/tts/`，避免重复请求。

密钥配置文件：

```text
ai_keys.local.json
```

该文件已被 `.gitignore` 忽略。模板见：

```text
ai_keys.example.json
```

也可以使用环境变量：

```bash
DEEPSEEK_API_KEY=xxx
MIMO_API_KEY=xxx
```

如果没有配置 AI，前端会自动降级为纯文字、静态剧情和静态结局，不会卡死。

### 服务接口

- `GET /api/health`：返回 AI 可用状态。
- `POST /api/llm`：代理 LLM 对话请求。
- `POST /api/tts`：生成或读取缓存语音，返回 PCM16 音频流。

### 预热语音缓存

启动服务器后可预生成静态对白语音：

```bash
node scripts/prewarm.mjs
```

指定端口：

```bash
PORT=9000 node scripts/prewarm.mjs
```

## 操作方式

| 按键 | 功能 |
|------|------|
| `WASD` / 方向键 | 移动 |
| `Shift` | 奔跑 |
| `E` | 交互 / 拾取 / 开始对话 |
| `Space` | 对话文本确认 / 冲刺 / 战斗确认 / 攻击瞄准确认 |
| `Ctrl` | 对话中按住快进（Skip） |

| `I` | 背包 |
| `M` | 世界地图 |

## 系统说明

### 存档

- 自动存档：场景切换、要石激活、战斗胜利或宽恕、通关等节点。
- 手动存档：`F6` 打开存档菜单，支持 3 个手动槽位。
- 快速存读：`F4` 快速存档，`F9` 快速读档。
- 存档包含场景、坐标、flags、karma、收集物、敌人击败记录、要石、刻字、难度和小地图探索记录。

### 战斗

- 战斗菜单包含战斗、调查、诗词、宽恕。
- 攻击瞄准使用 `E` 或 `Space` 停下光标。
- 诗词大招需要先收集完整诗句，每场战斗限用一次。
- 部分敌人具有特殊弹幕，例如格式化者会临时删除已收集的诗词碎片。

### 难度

三档难度会影响敌人生命、弹幕密度、弹幕速度、SAN 伤害、SAN 上限和宽恕所需调查次数。

### 小地图

右上角小地图支持战争迷雾，显示已探索区域、玩家、目标、道具、要石、NPC 和敌人。

## 项目结构

```text
.
├─ index.html              # 主游戏入口
├─ intro_3d.html           # 序幕 3D 页面
├─ server.js               # 兼容入口
├─ server/                 # 静态服务与 AI 代理
├─ scripts/                # 工具脚本，如 TTS 预热
├─ src/
│  ├─ game.js              # 游戏主状态机
│  ├─ scenes.js            # 场景数据、敌人和交互点
│  ├─ render.js            # 总渲染入口
│  ├─ render/              # HUD、场景、对话、战斗、面板渲染
│  ├─ systems/             # 交互、对话、刻字等系统
│  ├─ data/                # 对话、谜题、场景切换文案
│  ├─ ai/                  # LLM/TTS 前端接入
│  ├─ battle.js            # 战斗系统
│  ├─ sidescroll.js        # 江堤横版关卡
│  └─ level3d.js           # 维度裂隙 3D 关卡
├─ tests/                  # Vitest 测试
├─ assets/                 # 音频等资源
├─ docs/                   # 设计与剧情文档
└─ cache/tts/              # TTS 缓存目录
```
